An agentic system for explaining how online text misleads

We built a multi-agent misinformation detection system using a hybrid approach that combines large language model agents with machine learning models to analyze news articles along 4 factuality factors (frequency heuristic, sensationalism, echo chamber, and malicious accounts) and classify the truthfulness of the article. Our goal is to help everyday users independently assess credibility instead of relying on disappearing centralized fact-checking programs.

In a web where fact-checking systems are being dismantled and generative AI is making it trivial to manufacture convincing misinformation, users need tools that make credibility transparent, interpretable, and actionable.

What we built

  • ● A three-layer multi-agent system in Google ADK that performs claim extraction, parallel factuality analysis, and score aggregation.
  • ● Four supervised models trained on LIAR-PLUS for frequency heuristic, sensationalism, malicious account, and naive realism.
  • ● A series of prompting experiments from zero-shot baselines to Fractal Chain-of-Thought (FCoT) reasoning with tool calling.
  • ● An interactive factuality evaluation interface that exposes the agents and models to end users.

Problem & motivation

Fact-checking programs at large platforms have been scaled back or shut down, even as generative AI makes it trivial to produce realistic but deceptive content. Traditional pipelines that rely on centralized labels and human fact-checkers cannot keep up with the volume, speed, and subtlety of modern misinformation.

We focus on linguistic and contextual mechanisms that make content misleading even when individual sentences are technically true: repetition as evidence, emotional framing, bandwagon rhetoric, and one-sided narratives that omit key context. Our target users are curious readers, journalists, and researchers who want interpretable signals about how a piece of text might be manipulating them.

Approach in one screen

Multi-agent architecture
We build our system in three layers using Google’s Agent Development Kit: (1) a claim extraction and verification pipeline augmented with web search, (2) a parallel factuality analysis layer with four specialized LLM agents, and (3) a sequential aggregation layer that merges scores into an overall credibility assessment.
See: Methods Multi-agent ADK
Supervised factuality models
Using the LIAR-PLUS dataset (12.8K fact-checked political statements plus scraped PolitiFact extensions), we train four supervised models to score frequency heuristic, sensationalism, malicious accounts, and naive realism from linguistic features and metadata. These models are exposed to the agents as callable tools.
See: Data ML as tools
Prompting & reasoning design
We run a sequence of prompting experiments: zero-shot baselines, Chain-of-Thought, In-Context Learning with ten examples, tool calling, and four iterations of Fractal Chain-of-Thought (FCoT) with explicit objective functions for coverage and hallucination control.
See: Experiments Prompt engineering
Evaluation & limitations
On our 40-article ground truth set, structured prompts consistently beat naive baselines, especially for Malicious Account detection. However, Naive Realism remains challenging, and FCoT introduces higher compute costs due to multi-iteration reasoning. We discuss these trade-offs and failure modes in detail.
See: Results Limitations

So what? How to use this

Our system is not a binary “true / false” oracle. Instead, it surfaces multiple interpretable signals about how a piece of text may mislead and explains why it assigned each score. This is intended to:

  • Help readers spot rhetorical tricks like repetition-as-evidence and emotionally charged framing.
  • Give journalists and researchers a diagnostic lens for comparing outlets, articles, or time periods.
  • Provide a template for building safer LLM-based evaluators that emphasize structured reasoning, calibration, and tool grounding.
Repo & reproducibility

All code, models, and experiment scripts are available in our public GitHub repository: Multi-Agent-Misinformation-Analysis-System. Please see the repository README for up-to-date installation and demo instructions.
